# MISS-Frontend

Panel interno de gestión MISS.

## Tecnologías

- Angular 21 (standalone)
- Angular Material
- Bootstrap (grid + utilities)
- SCSS

## Estructura

- `src/app/core` — servicios, guards, interceptores, modelos
- `src/app/shared` — componentes reutilizables, pipes, directivas
- `src/app/features` — módulos por feature (login, dashboard, products, etc.)
